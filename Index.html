  • git clone - pass to repository.
  • git status - статус текущего локального репозитория
  • git add - добавить новый файл под контроль гита
  • git commit -am "added new file index.html" - зафиксировать изменения и дать им комментарий
  • git push - отправить все запрашиваемые изменения в репозиторий на сервер
  • git pull - забрать свежие обновления
